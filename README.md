**Kuriboh** is a simple and  shellcode loader written in Rust, designed to load and execute shellcodes.

## Requirements

- Rust (stable)
- Compatible operating system (Windows, Linux, etc.)

## How to use
```bash
cargo build --release
cargo run --release <path_to_shellcode.bin>
```
